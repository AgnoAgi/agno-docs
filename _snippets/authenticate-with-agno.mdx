### Authenticate with agno

Authenticate with agno by running the following command:

```shell
ag auth
```

or by exporting the `PHI_API_KEY` for your workspace from [agno.app](https://www.agno.app)

<CodeGroup>
```bash Mac
export PHI_API_KEY=phi-***
```

```bash Windows
setx PHI_API_KEY phi-***
```
</CodeGroup>
