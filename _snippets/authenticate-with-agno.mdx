### Authenticate with agno

Authenticate with agno.com by running the following command:

```shell
ag setup
```

or by exporting the `AGNO_API_KEY` for your workspace from [app.agno.com](https://www.app.agno.com)

<CodeGroup>
```bash Mac
export AGNO_API_KEY=ag-***
```

```bash Windows
setx AGNO_API_KEY ag-***
```
</CodeGroup>
