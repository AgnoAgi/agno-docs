## Delete local resources

Play around and stop the workspace using:

<CodeGroup>

```bash terminal
ag ws down
```

```bash full options
ag ws down --env dev --infra docker
```

```bash shorthand
ag ws down dev:docker
```

</CodeGroup>

or stop individual Apps using:

<CodeGroup>

```bash app
ag ws down --group app
```

```bash api
ag ws down --group api
```

```bash database
ag ws down --group db
```

</CodeGroup>
