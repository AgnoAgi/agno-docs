| Parameter                     | Type                                   | Default    | Description                                                    |
| ----------------------------- | -------------------------------------- | ---------- | -------------------------------------------------------------- |
| `id`                          | `str`                                  | -          | ID of the model to use. Alias: "model"                         |
| `name`                        | `Optional[str]`                        | -     | Name for this Model. Not sent to the Model API.                |
| `provider`                    | `Optional[str]`                        | -     | Provider for this Model. Not sent to the Model API.            |
| `metrics`                     | `Dict[str, Any]`                       | `{}`       | Metrics collected for this Model. Not sent to the Model API.   |
| `response_format`             | `Optional[Any]`                        | -     | Format of the response.                                        |
| `tools`                       | `Optional[List[Union[Tool, Dict]]]`    | -     | A list of tools provided to the Model.                         |
| `tool_choice`                 | `Optional[Union[str, Dict[str, Any]]]` | -     | Controls which (if any) function is called by the model.       |
| `show_tool_calls`             | `Optional[bool]`                       | -     | If True, shows function calls in the response.                 |
| `tool_call_limit`             | `Optional[int]`                        | -     | Maximum number of tool calls allowed.                          |
| `_functions`                  | `Optional[Dict[str, Function]]`        | -     | Functions extracted from the tools. Not sent to the Model API. |
| `_function_call_stack`        | `Optional[List[FunctionCall]]`         | -     | Function call stack.                                           |
| `system_prompt`               | `Optional[str]`                        | -     | System prompt from the model added to the Agent.               |
| `instructions`                | `Optional[List[str]]`                  | -     | Instructions from the model added to the Agent.                |
| `session_id`                  | `Optional[str]`                        | -     | Session ID of the calling Agent or Workflow.                   |
| `structured_outputs`          | `Optional[bool]`                       | -     | Whether to use the structured outputs with this Model.         |
| `supports_structured_outputs` | `bool`                                 | `False`    | Whether the model supports structured outputs.                 |
| `override_system_role`        | `bool`                                 | `False`    | Whether to override the system role.                           |
| `system_message_role`         | `str`                                  | `"system"` | The role to map the system message to.                         |
