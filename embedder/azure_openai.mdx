The `AzureOpenAIEmbedder` class is used to embed text data into vectors using the Azure OpenAI API.

## Usage

```python

import os
from typing import List
from rich.pretty import pprint  # noqa
from pydantic import BaseModel, Field

from dotenv import load_dotenv

from phi.vectordb.pgvector import PgVector
from phi.model.azure import AzureOpenAIChat
from phi.embedder.azure_openai import AzureOpenAIEmbedder
from phi.agent import Agent, AgentKnowledge, RunResponse  # noqa

load_dotenv()

embedding_model = AzureOpenAIEmbedder(
    model=os.getenv("AZURE_OPENAI_MODEL_NAME"),
    api_key=os.getenv("AZURE_OPENAI_API_KEY"),
    azure_endpoint=os.getenv("AZURE_OPENAI_ENDPOINT"),
    azure_deployment=os.getenv("AZURE_OPENAI_DEPLOYMENT"),
)

# Create knowledge base
knowledge_base=AgentKnowledge(
    vector_db=PgVector(
        db_url=db_url,
        collection=embeddings_table,
        embedder=embedding_model,
    ),
    # 2 references are added to the prompt
    num_documents=2,
),

# Add information to the knowledge base
knowledge_base.load_text("The sky is blue")

azure_chat_model = AzureOpenAIChat(
    id=os.getenv("AZURE_OPENAI_MODEL_NAME"),
    api_key=os.getenv("AZURE_OPENAI_API_KEY"),
    azure_endpoint=os.getenv("AZURE_OPENAI_ENDPOINT"),
    azure_deployment=os.getenv("AZURE_OPENAI_DEPLOYMENT"),
)

# Add the knowledge base to the Agent
agent = Agent(
  model=azure_model,
  knowledge=knowledge_base
)

# Run the agent
agent.run("Within the knowledge base, what is the color of the sky?")
```

## Params

<ResponseField name="model" type="str" default="text-embedding-ada-002">
  The name of the model used for generating embeddings.
</ResponseField>
<ResponseField name="dimensions" type="int" default="1536">
  The dimensionality of the embeddings generated by the model.
</ResponseField>
<ResponseField
  name="encoding_format"
  type="Literal['float', 'base64']"
  default="float"
>
  The format in which the embeddings are encoded. Options are "float" or
  "base64".
</ResponseField>
<ResponseField name="user" type="str">
  The user associated with the API request.
</ResponseField>
<ResponseField name="api_key" type="str">
  The API key used for authenticating requests.
</ResponseField>
<ResponseField name="api_version" type="str" default="2024-02-01">
  The version of the API to use for the requests.
</ResponseField>
<ResponseField name="azure_endpoint" type="str">
  The Azure endpoint for the API requests.
</ResponseField>
<ResponseField name="azure_deployment" type="str">
  The Azure deployment name for the API requests.
</ResponseField>
<ResponseField name="base_url" type="str">
  The base URL for the API endpoint.
</ResponseField>
<ResponseField name="azure_ad_token" type="str">
  The Azure Active Directory token for authentication.
</ResponseField>
<ResponseField name="azure_ad_token_provider" type="Any">
  The provider for obtaining the Azure AD token.
</ResponseField>
<ResponseField name="organization" type="str">
  The organization associated with the API request.
</ResponseField>
<ResponseField name="request_params" type="Optional[Dict[str, Any]]">
  Additional parameters to include in the API request. Optional.
</ResponseField>
<ResponseField name="client_params" type="Optional[Dict[str, Any]]">
  Additional parameters for configuring the API client. Optional.
</ResponseField>
<ResponseField name="openai_client" type="Optional[AzureOpenAIClient]">

</ResponseField>
