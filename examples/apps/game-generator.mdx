**GameGenerator** creates HTML5 games based on user descriptions using the Agno framework.

### Key Capabilities
- Generate single-page HTML5 games from user prompts
- Supports multiple AI models for game generation
- Stores game sessions in a PostgreSQL database
- Interactive Streamlit UI for game creation

### Example Game Descriptions
- "Create a classic Snake game with increasing speed and score display."
- "Develop a breakout clone with colorful blocks and multiple levels."
- "Design a space invaders game with retro graphics and sound effects."

## Usage

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/agno-api/agno.git
    cd agno
    ```
  </Step>

  <Step title="Create a virtual environment">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    pip install -r cookbook/examples/apps/game_generator/requirements.txt
    ```
  </Step>

  <Step title="Run PgVector">
    First, install [Docker Desktop](https://docs.docker.com/desktop/install/mac-install/).

    Then run either using the helper script:
    ```bash
    ./cookbook/run_pgvector.sh
    ```

    Or directly with Docker:
    ```bash
    docker run -d \
      -e POSTGRES_DB=ai \
      -e POSTGRES_USER=ai \
      -e POSTGRES_PASSWORD=ai \
      -e PGDATA=/var/lib/postgresql/data/pgdata \
      -v pgvolume:/var/lib/postgresql/data \
      -p 5532:5432 \
      --name pgvector \
      agnohq/pgvector:16
    ```
  </Step>

  <Step title="Set up API keys">
    ```bash
    # Required
    export OPENAI_API_KEY=***

    # Optional
    export ANTHROPIC_API_KEY=***
    export GOOGLE_API_KEY=***
    export GROQ_API_KEY=***
    ```
    We recommend using gpt-4o for optimal performance.
  </Step>

  <Step title="Launch the app">
    ```bash
    streamlit run cookbook/examples/apps/game_generator/app.py
    ```
    Open [localhost:8501](http://localhost:8501) to start using the Game Generator.
  </Step>

</Steps>
