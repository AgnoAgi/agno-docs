Use the best in class GPT models using Azure's OpenAI API.

## Authentication

Set your `AZURE_OPENAI_API_KEY` and `AZURE_OPENAI_ENDPOINT` environment variables.

<CodeGroup>

```bash Mac
export AZURE_OPENAI_API_KEY=***
export AZURE_OPENAI_ENDPOINT=***
# Optional:
# export AZURE_OPENAI_API_VERSION=***
# export AZURE_DEPLOYMENT=***
```

```bash Windows
setx AZURE_OPENAI_API_KEY ***
setx AZURE_OPENAI_ENDPOINT ***
# Optional:
# setx AZURE_OPENAI_API_VERSION ***
# setx AZURE_DEPLOYMENT ***
```

</CodeGroup>

## Example

Use `AzureOpenAIChat` with your `Agent`:

<CodeGroup>

```python agent.py
"""Run `pip install yfinance` to install dependencies."""

import os
from typing import Iterator  # noqa

from dotenv import load_dotenv

from phi.agent import Agent, RunResponse  # noqa
from phi.model.azure import AzureOpenAIChat
from phi.tools.yfinance import YFinanceTools

load_dotenv()

azure_model = AzureOpenAIChat(
    id=os.getenv("AZURE_OPENAI_MODEL_NAME"),
    api_key=os.getenv("AZURE_OPENAI_API_KEY"),
    azure_endpoint=os.getenv("AZURE_OPENAI_ENDPOINT"),
    azure_deployment=os.getenv("AZURE_OPENAI_DEPLOYMENT"),
)

agent = Agent(
    model=azure_model,
    tools=[YFinanceTools(stock_price=True)],
    show_tool_calls=True,
    markdown=True,
)

# Get the response in a variable
# run: RunResponse = agent.run("What is the stock price of NVDA and TSLA")
# print(run.content)

# Print the response on the terminal
agent.print_response("What is the stock price of NVDA and TSLA")
```

</CodeGroup>

## Params

<ResponseField name="name" type="str" default="AzureOpenAIChat">
  The name identifier for the agent.
</ResponseField>
<ResponseField name="id" type="str" required="true">
  The specific model ID used for generating responses. This field is required.
</ResponseField>
<ResponseField name="api_key" type="str">
  The API key for authenticating requests to the Azure OpenAI service.
</ResponseField>
<ResponseField name="api_version" type="str" default="2023-12-01-preview">
  The version of the Azure OpenAI API to use.
</ResponseField>
<ResponseField name="azure_deployment" type="str">
  The deployment name or ID in Azure.
</ResponseField>
<ResponseField name="base_url" type="str">
  The base URL for making API requests to the Azure OpenAI service.
</ResponseField>
<ResponseField name="azure_ad_token" type="str">
  The Azure Active Directory token for authenticating requests.
</ResponseField>
<ResponseField name="azure_ad_token_provider" type="str">
  The provider for obtaining Azure Active Directory tokens.
</ResponseField>
<ResponseField name="openai_client" type="AzureOpenAIClient">
  An instance of AzureOpenAIClient provided for making API requests.
</ResponseField>
