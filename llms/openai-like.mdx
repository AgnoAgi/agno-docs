---
title: OpenAI Like
---

Many providers like Together and Groq support the OpenAI API format. Use the `OpenAILike` model to access them by replacing the `base_url`.

## Example

<CodeGroup>

```python agent.py
from os import getenv
from phi.agent import Agent
from phi.model.openai.like import OpenAILike

agent = Agent(
    model=OpenAILike(
        id="mistralai/Mixtral-8x7B-Instruct-v0.1",
        api_key=getenv("TOGETHER_API_KEY"),
        base_url="https://api.together.xyz/v1",
    )
)
agent.print_response('Share a 2 sentence horror story.', markdown=True)
```

</CodeGroup>

## Params

<ResponseField name="id" type="str">
  The name of the model to be used for generating responses.
</ResponseField>
<ResponseField name="api_key" type="str">
  The API key for authenticating requests to the service.
</ResponseField>
<ResponseField name="base_url" type="str">
  The base URL for making API requests to the service.
</ResponseField>

`OpenAILike` also support all the params of [OpenAIChat](/llms/openai)
