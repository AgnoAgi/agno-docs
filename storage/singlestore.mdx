Phidata supports using SingleStore as a storage backend for Agents using the `PgAgentStorage` class.

## Usage

```python storage.py
from phi.storage.agent.singlestore import S2AgentStorage

# Create a storage backend using the SingleStore database
storage = S2Storage(
    # store runs in the ai.agent_runs table
    table_name="agent_runs",
    # db_url: SingleStore database URL
    db_url=db_url,
)

# Add storage to the Agent
agent = Agent(storage=storage)
```

## S2AgentStorage Params

<ResponseField name="table_name" type="str">
  The name of the table to store agent conversations in.
</ResponseField>
<ResponseField name="schema" type="str" default="llm">
  The schema to store the table in.
</ResponseField>
<ResponseField name="db_url" type="str">
  The database URL to connect to.
</ResponseField>
<ResponseField name="db_engine" type="Engine">
  The database engine to use.
</ResponseField>
